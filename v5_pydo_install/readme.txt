TOUT_DOUX_LISTE version 5


Le programme a été créé en Python avec une base de donnée SQLite. Il a été transformé en .exe via pyinstaller puis en programme installable grâce à Inno Setup.


Le programme s'installe dans le dossier Programme(X86).

Le fichier de la base de donnée s'installe ici : C:\Users\stagiaire\AppData\Roaming\ToutDouxListe
